Sales Report Dashboard

Project Overview

The Sales Report Dashboard is a data visualization project that provides key insights into total sales, profit, and units sold. It helps businesses monitor performance, identify best-selling products, and analyze sales trends over time.

The dashboard includes:
[Sales Report Dashboard.docx](https://github.com/user-attachments/files/23419532/Sales.Report.Dashboard.docx)

•	Top 10 Product Sales — Highlights top-performing products.
•	Quantity of Sales by Type — Compares Online, Retail, and Wholesaler sales.
•	Sales Type Distribution — Displays percentage breakdown by sales category.
•	Monthly Revenue Trend — Shows revenue growth and seasonal variations.
 
Dashboard Highlights
•	Real-time tracking of business KPIs.
•	Comparison of product and sales channel performance.
•	Insightful visual trends for better business decisions.
